Project Milestones

Milestones of the job upfront:

  • Write down what the client wants
  • Write down what the law requires
  • Combine those into a project plan
  • Add your own thoughts to the project plan

Setting up:

  • Use (s)FTP to move files to a web server
  • Install a content management system
  • Set up the content management system
  • Lock the website to have dev access only

Basic Development:

  • Select a frontend framework
  • Create a basic CSS file
  • Create sitewide structure:
    • Create a menu
    • Create headers
    • Create machine-readable elements (schema.org / ogp.me)
  • Create page structure
  • Create single elements

Finetuning:

  • Create a second set of layouts (responsive layout)
  • Create OG images

Checking:

  • Check the entire site in a validator
  • Check site display in search engines
  • Check link preview in Facebook / Twitter
  • Check layout on different devices

Hardening:

  • Move core folders of CMS out of the public internet
  • Check SSL
  • Deactivate all admin accounts but one

End of Project
